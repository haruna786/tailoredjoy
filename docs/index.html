丁寧な作品の中に、
楽しさや遊び心を。

一人ひとりの好きや素敵という感性に寄り添い、喜びを丁寧に仕立てるという想いを込めています。単なる情報ではなく、あなたが持つ心地よい雰囲気や醸し出す空気感を読み取り、細部まで丁寧に仕立てていきたいです。その中で楽しさや遊び心を忘れずにデザインに向き合っていきます。

wan

Haruna Ikeda
